# SyrnykSpotifybot

Telegram bot for Spotify listening stats, liked tracks publishing and more — powered by Spotipy

### Getting Started
 - create `.env` like a `.env.example`
 - load libraries from requirements
 ```powershell
 pip install -r requirements.txt
 ```
 - in terminal do:
```powershell
python bot.py
```
### Commands
- `/start` — start bot
- `/help` — list of commands
- `/login` — connect Spotify
- `/status` — check authorization

### Technologies
* Python
* aiogram 3
* Spotipy
